Aprendendo HTML5

Aula 1 

 
    >!DOCTYPE html>  ----------- Declara o tipo do documento e informa ao navegador que o arquivo utiliza HTML5
    >html lang="pt-br">  -------- Definir o idioma principal da página como português do Brasil, auxiliando na acessibilidade e na indexação por mecanismos de busca
    >head>  -------------- Seção de cabeçalho que contém metadados e configurações gerais do site
    >meta charset="UTF-8"/>  ------- Definir o conjunto de caracteres como UTF-8, permitindo o uso correto de acentos e caracteres especiais. Essa tag não possui fechamento
    >title> Qualquer título aqui >/title>  -------- Defina o título da página, exibido na aba do navegador 
    >style>  ------ Área destinada ao CSS, responsável pela estilização e formatação visual da página. Por convenção, o CSS é declarado no cabeçalho
    h1 {  ----- Seletor CSS que aplica estilos a todos os elementos >h1>. As propriedades definidas entre as chaves controlam a aparência do título
        font-family: Arial;  ------ Definir uma família tipográfica utilizada no texto 
        font-size: 30pt;  --------- Defina o tamanho da fonte
        color: blue;  --------- Defina uma cor do texto
        text-shadow: 2px 2px 2px blue;  ------- Aplicar sombra ao texto, seguindo a ordem: deslocamento horizontal, deslocamento vertical, nível de desfoque e cor da sombra
    } 
    >/style>  -------- Finaliza a seção de estilo do documento
    >/head> ------- Finaliza a seção de cabeçalho do documento
    >body> --------------- Corpo do documento, onde fica todo o conteúdo visível ao usuário 
    >h1> Olá mundo! >/h1>  ------------ Título de nível 1, utilizado para títulos principais da página
    >h2> Me livrando da madição >/h2>  -------------- Título de nível 2, utilizado para legendas
    >h1> OBA! Me livrei! >/h1>  ---------------- Outro título de nível 1; para compartilhar a mesma tag, herda automaticamente a formatação CSS definida anteriormente
    >/body>  --------------- Finalize o conteúdo visível da página
    >/html>  ----------- Encerra a estrutura do documento HTML
